# san-gulp-starter

## Prepare

```
$ npm install
```

## Run

```
npm run build
```

## features

### JS

- TypeScript
- babel
- AMD
- concat
- hash

### CSS

- concat
- hash

### San

- `.san.ts` + `.html` + `.css`
- Auto inline `.html` in .san.ts:
    ```javascript
    import template from 'xxx.html!text';
    ```

### HTML

- Compile to `.tpl`.
- Resource filename replace (with hash).
- Inline Resource.


## How to open


Use any HTTP Server to open this example. Such as:

```
npm run dev
```

and open http://172.24.3.24:8080 use any browser.
